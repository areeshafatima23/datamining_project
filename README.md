# News Classification Project

## Description
This project implements a news classification pipeline using TF-IDF, PCA, SVM, and Random Forest.

## Features
- Text preprocessing (NLTK)
- Feature engineering (TF-IDF + PCA)
- SVM and Random Forest models
- Confusion matrices & word clouds
- Streamlit dashboard for real-time predictions

## Dataset
The dataset used is the [Combined News Dataset](https://www.kaggle.com/datasets/areeshafatimaaf/combined-news-dataset). 
This is a combined version of the Dawn and Tribune Datasets used for our NLP model.
